>>  #### بررسی میزان حافظه اشغال شده
>>  **دستور `du` (disk usage)**


##### • حافظه اشغال شده در مسیر پوشه فعلی
```bash
du
```
‌
##### •  حافظه اشغال شده در مسیر مشخص 
```bash
du /home/files/videos
```

| --------------------- | --- |

| -a, --all (`du -ah`)  |     |

| --time                |     |‌‌‌
